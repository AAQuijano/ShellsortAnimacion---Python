## Descripción
Animación visual del algoritmo de ordenamiento ShellSort implementada en Python usando Tkinter.

## Requisitos
- Python 3.x
- Tkinter (incluido en Python)

## Uso
1. Ejecutar el archivo `Proyecto_Final.py`
2. Agregar valores al arreglo (máximo 11 valores)
3. Hacer clic en "Ordenar Arreglo" para ver la animación del ordenamiento ShellSort
4. Usar "Nuevo Arreglo" para reiniciar

## Controles
- **Agregar valor al Arreglo**: Ingresa un número al arreglo
- **Ordenar Arreglo**: Inicia la animación del algoritmo ShellSort
- **Nuevo Arreglo**: Limpia el arreglo actual
